  1. 访问 管理登录页面
  2. 使用管理员账号登录 (用户名: admin, 密码: admin123)
  3. 进入用户管理页面
  4. 点击任意用户的"编辑"按钮

    • 从教师切换到学生

  • ✅ 用户类型选择框可以正常切换
