==========================================
依赖检查方法
==========================================

### 如何查找项目依赖？

```bash
# 方法1：扫描import语句
grep -r "^import " --include="*.py" | awk '{print $2}' | sort -u

# 方法2：使用pipreqs工具
pip install pipreqs
pipreqs .

# 方法3：检查导入的标准库和第三方库
python -c "import sys; import pip; installed = pip.get_installed_distributions(); print([pkg.key for pkg in installed])"
```

### 本次使用的检查方法

```bash
# 扫描所有Python文件中的导入
grep -r "^import \|^from " backend/app/ --include="*.py" | \
  grep -v "from \." | \
  grep -v "from app" | \
  grep -v "from backend" | \
  awk '{print $2}' | \
  cut -d'.' -f1 | \
  sort -u
```
