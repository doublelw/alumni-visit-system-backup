# 管理后台数据统计功能说明

## 概述

校友入校登记系统管理后台提供了丰富的数据统计功能，支持多维度、多时间段的数据分析和报表生成。

## 统计功能分类

### 1. 概览统计 (`/api/statistics?type=overview`)

提供系统整体运行情况的统计概览，包括：

#### 时间段访问统计
- **当日访问量**: 当天所有入校记录数量
- **本周访问量**: 本周内所有入校记录数量
- **本月访问量**: 本月内所有入校记录数量
- **本年度访问量**: 本年度内所有入校记录数量
- **总访问量**: 系统上线以来的总访问记录数量

#### 用户状态统计
- **总用户数**: 系统中注册的所有用户数量
- **校友数量**: 已认证的校友用户数量
- **教师数量**: 教师账户数量
- **保安数量**: 保安账户数量

#### 申请状态统计
- 申请按状态分类统计（待审核、已通过、已拒绝、已完成等）

#### 院系分布统计
- 按校友所在院系进行分类统计，显示校友分布情况

#### 年级分布统计
- 按校友毕业年份进行分类统计，了解不同年级校友活跃度

### 2. 访问详细统计 (`/api/statistics?type=visits`)

针对访问行为的深度统计分析：

#### 受访人统计
- 统计各受访人员的访问次数
- 识别最受欢迎的受访对象
- 排行榜形式展示Top 20受访人

#### 访问人统计
- 统计各校友的访问频次
- 识别最活跃的校友用户
- 排行榜形式展示Top 20访问人

#### 审批人统计
- 统计各审批人员处理的申请数量
- 评估审批工作效率
- 按审批量排序展示

#### 放行人统计
- 统计各保安人员的放行记录
- 门卫工作量统计
- 保安工作绩效评估

#### 访问目的统计
- 按访问目的分类统计（学术交流、参观校园、拜访老师等）
- 了解主要访问动机
- 为资源配置提供数据支持

### 3. 人员统计 (`/api/statistics?type=people`)

针对校友群体的多维度分析：

#### 班级统计
- 按校友所在班级进行统计
- 识别最活跃的班级群体
- 班级活跃度排行

#### 年级-班级统计
- 年级与班级的交叉分析
- 细粒度的群体活跃度分析
- 支持年级对比和班级对比

### 4. 车辆统计 (`/api/statistics?type=vehicles`)

车辆管理相关的统计分析：

#### 车辆状态统计
- 按车辆注册状态分类（待审核、已通过、已拒绝等）
- 车辆申请通过率分析

#### 车辆品牌统计
- 按车辆品牌分类统计
- 了解校友车辆品牌偏好
- 车辆品牌分布情况

#### 车辆颜色统计
- 按车辆颜色分类统计
- 车辆颜色分布情况

## API使用示例

### 获取概览统计
```bash
GET /api/statistics?type=overview
```

### 获取访问统计
```bash
GET /api/statistics?type=visits
```

### 获取人员统计
```bash
GET /api/statistics?type=people
```

### 获取车辆统计
```bash
GET /api/statistics?type=vehicles
```

## 数据更新频率

- **实时数据**: 当日访问量、用户总数等实时更新
- **定时汇总**: 小时、日、周、月统计数据定时更新
- **缓存机制**: 常用统计数据进行缓存，提高查询效率

## 统计图表支持

系统支持多种图表展示方式：
- **折线图**: 访问趋势分析
- **柱状图**: 分类数据对比
- **饼图**: 比例关系展示
- **热力图**: 时间分布分析

## 数据导出功能

支持统计数据导出：
- **Excel格式**: 详细数据表格导出
- **PDF报告**: 图表和统计数据整合报告
- **CSV格式**: 原始数据导出

## 隐私保护

- 统计数据脱敏处理
- 个人信息聚合展示
- 符合数据保护法规要求

## 使用建议

1. **定期查看**: 建议每周查看访问趋势统计
2. **重点监控**: 关注异常访问模式和高峰时段
3. **数据驱动**: 基于统计数据优化资源配置
4. **安全预警**: 通过统计发现潜在安全风险

## 技术特点

- **高性能**: 数据库查询优化，支持大数据量统计
- **实时性**: 关键指标实时更新
- **可扩展**: 支持自定义统计维度
- **准确性**: 数据校验机制确保统计准确性
