# 校友入校登记系统 - 快速开始指南

## 🚀 服务器管理

### 首次使用

1. **安装依赖**
   ```bash
   # 方法1: 使用我们的自动安装脚本
   setup_and_start.bat

   # 方法2: 手动安装
   pip install -r requirements.txt
   ```

2. **启动服务器**
   ```bash
   # Windows快速重启（推荐）
   双击运行 quick_restart.bat

   # 或者使用命令行
   python server_manager.py restart --debug --app-dir backend
   ```

### 常用命令

```bash
# 重启服务器（开发模式）
python server_manager.py restart --debug --app-dir backend

# 启动服务器
python server_manager.py start --debug --app-dir backend

# 停止服务器
python server_manager.py stop --app-dir backend

# 查看服务器状态
python server_manager.py status --app-dir backend

# 强制停止所有进程
python server_manager.py stop --force --app-dir backend

# 启动生产模式服务器
python server_manager.py start --env production --app-dir backend
```

## 📱 访问应用

- **开发环境**: http://127.0.0.1:5000
- **前端应用**: http://127.0.0.1:5000
- **管理后台**: http://127.0.0.1:5000/admin

## 🔧 故障排除

### 1. 依赖问题
```bash
# 重新安装依赖
python server_manager.py install --app-dir backend

# 或者手动安装
pip install -r requirements.txt
```

### 2. 端口占用
```bash
# 查看状态
python server_manager.py status --app-dir backend

# 强制停止
python server_manager.py stop --force --app-dir backend
```

### 3. 应用无法启动
```bash
# 检查日志
type backend\logs\server_*.log

# 或者查看详细错误
python server_manager.py start --debug --app-dir backend
```

## 💡 使用技巧

1. **开发时建议**: 使用 `--debug` 参数启用自动重载
2. **生产环境**: 使用 `--env production` 参数
3. **查看日志**: 检查 `backend/logs/` 目录下的日志文件
4. **快速操作**: 使用 `quick_restart.bat` 进行快速重启

## 📞 获取帮助

如遇到问题，请检查：
1. Python环境是否正确安装
2. 依赖是否完整安装
3. 端口5000是否被占用
4. 查看日志文件获取详细错误信息
