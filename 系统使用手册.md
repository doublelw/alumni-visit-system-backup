# 校友入校登记系统使用手册

## 📖 系统概述

校友入校登记系统是一个集校友信息管理、访问申请审批、人脸识别验证、车辆管理、数据统计分析于一体的综合性校园管理系统。系统采用现代化的Web技术架构，支持多终端访问，为不同角色的用户提供便捷高效的服务。

### 🎯 系统特性

- **多角色权限管理** - 支持校友、教师、保安、管理员四种角色
- **智能人脸识别** - 快速准确的身份验证
- **移动端适配** - 完美适配手机和平板设备
- **实时数据统计** - 全面的访问数据分析
- **批量授权功能** - 高效的群体访问管理
- **安全加密传输** - 全程HTTPS加密保护

### 🌐 访问地址

- **移动端/访客界面**: https://localhost:5000
- **管理后台**: https://localhost:5000/admin
- **API接口**: https://localhost:5000/api/*

## 🏢 系统架构与技术栈

### 后端技术
- **Python 3.7+** - 主要开发语言
- **Flask** - Web应用框架
- **SQLAlchemy** - ORM数据库操作
- **JWT** - 用户认证授权
- **MySQL/SQLite** - 数据存储
- **OpenCV** - 人脸识别处理

### 前端技术
- **HTML5 + CSS3** - 响应式布局设计
- **JavaScript ES6+** - 现代化交互逻辑
- **Chart.js** - 数据可视化图表
- **Remix Icon** - 图标库系统

### 安全特性
- **HTTPS加密** - SSL/TLS安全传输
- **权限控制** - 细粒度角色权限管理
- **数据加密** - 敏感信息加密存储
- **人脸保护** - 生物特征安全处理

   │  真实姓名: __________ │
   │  邮箱: _____________ │
   │  手机号: ___________ │
   │  密码: _____________ │
   │  确认密码: _________ │
   └─────────────────────┘
   ```

3. **完善校友档案**
   ```
   ┌─────────────────────┐
   │  毕业年份: 2020     │
   │  学号: 2020XXXX     │
   │  班级: 计算机科学1班│

   │  紧急联系人: _____ │
   │  紧急联系电话: ___ │

   ```

4. **提交审核**
   - 信息填写完成后提交
   - 等待管理员/教师审核
   - 审核通过后即可正常使用系统

3. **访问人信息**
   ```
   ┌─────────────────────────┐
   │  访问人姓名: 张三 *      │
   │  联系电话: 138********  │
   │  邮箱地址: _________    │
   │  身份证号: __________*  │

   ```

4. **同行人信息（可选）**
   - 点击"添加同行人"按钮
   - 填写同行人姓名和联系方式
   - 可添加多个同行人

   - 可在"申请历史"中查看进度

### 🔍 故障排除

#### 常见问题解决

1. **系统启动失败**
   ```
   问题: 服务器启动失败
   检查清单:
   ☑️ Python版本 >= 3.7
   ☑️ 依赖包是否完整安装
   ☑️ 数据库连接是否正常
   ☑️ 端口5000是否被占用
   ☑️ SSL证书文件是否存在

   解决方案:
   1. 重新安装依赖: pip install -r requirements.txt
   2. 检查数据库配置
   3. 更换端口或关闭占用进程
   4. 重新生成SSL证书
   ```

2. **数据库连接问题**
   ```
   问题: 无法连接到数据库
   检查清单:
   ☑️ 数据库服务是否运行
   ☑️ 连接配置是否正确
   ☑️ 用户权限是否足够
   ☑️ 网络连接是否正常

   解决方案:
   1. 启动数据库服务
   2. 检查config/prod.py配置
   3. 验证数据库用户权限
   4. 测试网络连通性
   ```

3. **人脸识别失败**
   ```
   问题: 人脸识别准确率低

   1. 改善光线条件

   4. 重新注册高质量人脸图像
