# 微信云托管部署指南

## 📋 部署清单

### ✅ 1. 数据库准备（重要！）

微信云托管**不支持SQLite**，需要使用云数据库：

#### 选项A：腾讯云MySQL（推荐）
1. 登录腾讯云控制台
2. 创建云数据库 MySQL
3. 获取数据库连接信息：
   - 主机地址
   - 端口（默认3306）
   - 数据库名
   - 用户名
   - 密码

#### 选项B：微信云数据库
1. 在微信云托管控制台
2. 开通云数据库服务
3. 创建数据库实例

---

### ✅ 2. 修改数据库配置

**文件位置**: `backend/app/config.py`

```python
class ProductionConfig(Config):
    DEBUG = False
    TESTING = False

    # 使用腾讯云MySQL（替换为你的实际信息）
    SQLALCHEMY_DATABASE_URI = 'mysql+pymysql://username:password@host:port/database_name'

    # JWT密钥（生产环境必须修改）
    JWT_SECRET_KEY = 'your-production-secret-key-change-this'
    JWT_ACCESS_TOKEN_EXPIRES = 86400  # 24小时

    # 电子校友卡密钥
    ELECTRONIC_CARD_SECRET_KEY = 'your-card-secret-key'

    # CORS设置
    CORS_ORIGINS = '*'  # 生产环境建议设置具体域名
```

**更新 `backend/app/__init__.py`**:
```python
def create_app(config_name=None):
    # ... 现有代码 ...

    # 根据环境变量选择配置
    if config_name is None:
        # 优先使用环境变量
        if os.environ.get('WECHAT_CLOUD'):
            config_name = 'production'
        else:
            config_name = os.environ.get('FLASK_ENV', 'development')

    # 加载配置
    from app.config import config
    Config = config.get(config_name, config['default'])
```

### ✅ 3. 环境变量配置

在微信云托管后台设置以下环境变量：

```bash
FLASK_APP=app.py
FLASK_ENV=production
SECRET_KEY=your-production-secret-key
DATABASE_URL=mysql+pymysql://user:pass@host:port/db
WECHAT_CLOUD=true
JWT_SECRET_KEY=your-jwt-secret
ELECTRONIC_CARD_SECRET_KEY=your-card-secret
```

### ✅ 5. 部署步骤

#### **方式1：微信云托管控制台部署**

1. 登录微信公众平台
2. 进入"微信云托管"控制台
3. 创建服务 → 选择 **Flask** 框架
4. 上传代码包或连接Git仓库
5. 配置环境变量
6. 配置云数据库
7. 启动服务

#### **方式2：使用Git仓库部署（推荐）**

1. 将项目推送到Git仓库（Gitee/GitHub）
2. 在微信云托管选择"从仓库导入"
3. 选择 **Flask** 框架
4. 配置环境变量和数据库
5. 部署

## 🆘 常见问题

### 1. 数据库连接失败
**原因**: 云数据库连接信息错误或防火墙
**解决**:
- 检查数据库白名单
- 验证连接字符串
- 确认数据库已启动

### 2. 依赖安装失败
**原因**: requirements.txt中的包不兼容
**解决**: 锁定版本号或使用腾讯云镜像源

### 3. 静态资源404
**原因**: static文件夹路径配置错误
**解决**: 检查`app/__init__.py`中的static_folder配置
