# 微信云托管部署操作步骤（当前页面版）

## ✅ 你已完成：成功登录微信云托管

当前页面：`https://cloud.weixin.qq.com/cloudrun/service`

---

## 🎯 现在开始创建服务并部署

### 步骤1：创建新服务

在当前页面（`/cloudrun/service`），你应该能看到：

#### 情况A：如果看到"新建服务"按钮
```
┌─────────────────────────────────────────┐
│  [+ 新建服务]  [刷新]                    │
└─────────────────────────────────────────┘
    ↑
    点击这个按钮
```

**操作**：
1. 点击 **"新建服务"** 或 **"+ 新建服务"** 按钮
2. 可能会弹出创建服务对话框

#### 情况B：如果看不到服务列表，直接显示创建界面
- 直接进行下一步

---

### 步骤2：填写服务信息

在创建服务页面，填写以下信息：

```
┌─────────────────────────────────────────┐
│  创建服务                                │
├─────────────────────────────────────────┤
│                                          │
│  服务名称: [lsalumni-api          ]     │
│                                          │
│  服务简介: [校友入校登记系统API    ]     │
│                                          │
│  部署方式:                                │
│    ○ 从代码库创建（Git）                 │
│    ● 本地上传代码包  ← 选这个             │
│                                          │
│  [下一步]                                │
│                                          │
└─────────────────────────────────────────┘
```

**填写建议**：
- **服务名称**：`lsalumni-api` 或 `alumni-visit-system`
- **服务简介**：`校友入校登记系统后端API`
- **部署方式**：选择 **"本地上传代码包"**

---

### 步骤3：上传代码包

#### 如果还没有部署包：

**打开命令行，运行部署脚本**：

**Windows**：
```bash
双击运行: 一键部署到微信云.bat
```

**或手动运行**：
```bash
python deployment_scripts/deploy_to_welife.py
```

脚本会生成：
- `welife_deploy_package_YYYYMMDD_HHMMSS.zip`

#### 回到云托管页面：

1. 点击 **"选择文件"** 或 **"上传"** 按钮
2. 选择刚才生成的 `.zip` 文件
3. 等待上传完成（显示上传进度条）

---

### 步骤4：配置环境变量（重要！）

上传完成后，进入环境变量配置页面：

#### 点击"环境变量"或"配置"标签

在配置页面添加以下环境变量：

```
┌─────────────────────────────────────────┐
│  环境变量配置                            │
├─────────────────────────────────────────┤
│                                          │
│  [+ 添加变量]                            │
│                                          │
│  变量名          变量值                   │
│  ─────────────────────────────────────  │
│  FLASK_APP       app.py                  │
│  FLASK_ENV       production              │
│  WECHAT_CLOUD    true                    │
│                                          │
│  [+ 继续添加更多变量...]                 │
│                                          │
└─────────────────────────────────────────┘
```

#### 必须配置的变量：

**基础配置**：
```
FLASK_APP = app.py
FLASK_ENV = production
WECHAT_CLOUD = true
```

**数据库配置**（暂时使用SQLite，测试通过后再改MySQL）：
```
DATABASE_URL = sqlite:///alumni_system.db
```

**安全密钥**（生成随机密钥）：
```
# 在本地运行生成密钥
openssl rand -base64 32

# 然后填入：
SECRET_KEY = [生成的密钥]
JWT_SECRET_KEY = [生成的密钥]
ELECTRONIC_CARD_SECRET_KEY = [生成的密钥]
HMAC_SECRET_KEY = [生成的密钥]
```

**完整环境变量列表**：
```
FLASK_APP=app.py
FLASK_ENV=production
WECHAT_CLOUD=true
DATABASE_URL=sqlite:///alumni_system.db
SECRET_KEY=你的密钥1
JWT_SECRET_KEY=你的密钥2
ELECTRONIC_CARD_SECRET_KEY=你的密钥3
HMAC_SECRET_KEY=你的密钥4
```

```

│                                          │
│  实例规格:                                │
│    ● 1C 2G（基础版）                     │
│      - 费用：约 ¥XXX/月                  │
│                                          │
│  实例数量: [1] 个                        │
│                                          │
│  [开始部署]                              │
│                                          │
└─────────────────────────────────────────┘
```

**推荐配置**：
- **实例规格**：选择最便宜的（1C 2G 或 0.5C 1G）
- **实例数量**：1个（测试阶段）
- **版本描述**：`v1.0.0 - 初始部署`

---

### 步骤6：开始部署

1. 点击 **"开始部署"** 或 **"立即部署"** 按钮
2. 会显示部署进度：
   ```
   ⏳ 正在构建镜像...
   ⏳ 正在部署实例...
   ⏳ 正在配置环境...
   ```

3. 等待2-5分钟，直到显示：
   ```
   ✅ 部署成功
   ```

## 🔧 部署后配置

### 1. 配置公众号服务器地址

回到微信公众平台：
```
开发 → 基本配置 → 服务器配置
```

填写：
- **URL**: `https://你的服务地址.welife.icu/api/wechat`
- **Token**: 自定义令牌
- **EncodingAESKey**: 自动生成

### 2. 配置自定义域名（可选）

如果有自己的域名：
```
云托管 → 服务 → 域名管理 → 添加域名
```

## ⚠️ 常见问题

### 1. 部署失败 - 数据库错误
**原因**：环境变量未配置或配置错误

**解决**：
- 检查环境变量是否正确添加
- 使用SQLite测试：`DATABASE_URL=sqlite:///alumni_system.db`

### 2. 服务启动失败
**原因**：启动命令或端口问题

**解决**：
- 检查Dockerfile中的CMD命令
- 确保使用环境变量PORT

### 3. 无法访问
**原因**：服务未启动或网络问题

**解决**：
- 查看服务日志
- 检查服务状态是否为"运行中"
