# 校园安全出入管理系统 - 功能说明文档

## 系统概述

校园安全出入管理系统是一个综合性的校园访问管理平台，集成了校友访问、外来人员登记、临时访问、学生请假离校等多种出入场景，并提供了完整的后台管理和数据分析功能。

### 核心特性
- 🔐 多重身份验证（人脸识别、二维码、验证码）
- 📱 移动端友好的响应式设计
- 🔄 实时审批流程和状态跟踪
- 📊 数据统计和报告生成
- 🎯 分级权限管理

## 1. 校友访问流程

### 1.1 流程描述
校友通过系统申请访问校园，需要经过档案审核和访问审批双重验证。

### 1.2 用户角色
- **校友**：申请访问者
- **管理员**：审核校友档案和访问申请
- **保安**：验证访客身份并登记出入

### 1.3 操作步骤

#### 校友端操作
1. **登录系统**
   - 使用手机号/密码登录
   - 或使用人脸识别登录

2. **完善校友档案**
   - 填写毕业信息（学号、班级、专业、毕业年份）
   - 提供班主任联系方式
   - 上传人脸照片（可选）

3. **提交访问申请**
   - 选择访问日期和时间段
   - 填写访问事由
   - **新增安全验证**：填写受访者工作ID/学号和姓名，点击验证按钮
   - 系统自动验证受访者信息
   - 填写访问人信息（如非本人访问）

4. **查看申请状态**
   - 实时查看审批进度
   - 查看审批意见
   - 获批准后生成二维码

#### 管理员端操作
1. **校友档案审核**
   - 审核校友提交的档案信息
   - 验证毕业信息的真实性
   - 批准或拒绝档案申请

2. **访问申请审批**
   - 审核访问事由和时间
   - 检查受访者信息
   - 批准或拒绝访问申请

#### 保安端操作
1. **验证访客身份**
   - 扫描访客二维码
   - 或输入验证码验证
   - 或使用人脸识别验证

2. **登记出入信息**
   - 确认访客身份后登记入校
   - 记录实际出入时间
   - 异常情况处理

## 2. 外来送货人员登记流程

### 2.1 流程描述
为校园送货、维修等临时外来人员提供快速登记通道。

### 2.2 用户角色
- **外来人员**：送货、维修等服务人员
- **保安**：登记外来人员信息

### 2.3 操作步骤

#### 外来人员登记
1. **身份登记**
   - 提供有效身份证件
   - 登记姓名和联系电话
   - 说明来访事由（送货、维修等）

2. **受访者确认**
   - 提供受访部门或人员信息
   - 保安电话确认受访者

3. **临时证件发放**
   - 发放临时访客证件
   - 记录出入时间

#### 保安端操作
1. **信息核实**
   - 核实外来人员身份
   - 确认受访者信息

2. **系统登记**
   - 在保安端系统中登记访客信息
   - 生成临时访问记录

3. **出入管理**
   - 监控访客在校活动
   - 记录离校时间

## 3. 临时访问流程

### 3.1 流程描述
为非校友的临时访客（如家长、商务访客等）提供访问申请服务。

### 3.2 用户角色
- **访客**：临时访问人员
- **受访者**：校园内被访问人员
- **保安**：验证访客身份

### 3.3 操作步骤

#### 访客申请
1. **在线申请**
   - 填写访客基本信息
   - 提供受访者信息（需验证）
   - 说明访问事由

2. **受访者确认**
   - 系统向受访者发送确认通知
   - 受访者确认或拒绝访问

3. **获取访问凭证**
   - 确认后生成访问二维码
   - 访客凭二维码入校

#### 保安端验证
1. **身份核对**
   - 扫描访客二维码
   - 核对访客身份信息

2. **受访者确认**
   - 联系受访者确认访问
   - 记录访问信息

## 4. 学生请假离校流程

### 4.1 流程描述
学生申请请假离校，需要家长和班主任双重审批。

### 4.2 用户角色
- **学生**：提交请假申请
- **家长**：审批请假申请
- **班主任**：最终审批
- **保安**：验证离校凭证

### 4.3 操作步骤

#### 学生申请
1. **提交请假申请**
   - 选择离校日期和时间
   - 填写请假事由
   - 填写离校目的地
   - 添加同行人信息（可选）

2. **查看审批状态**
   - 实时查看家长审批状态
   - 查看班主任审批状态
   - 查看审批意见

#### 家长审批
1. **人脸识别登录**
   - 使用人脸识别验证身份
   - 确保账号安全

2. **审批申请**
   - 查看孩子请假详情
   - 确认或拒绝请假申请
   - 填写审批意见

#### 班主任审批
1. **审核申请**
   - 查看学生请假信息
   - 确认家长审批结果
   - 最终审批决定

2. **特殊情况处理**
   - 需要补充材料的情况
   - 需要面谈的情况

#### 保安验证
1. **验证离校凭证**
   - 扫描学生二维码
   - 验证双重审批结果
   - 确认离校信息

2. **人脸识别验证**
   - 对学生进行人脸扫描
   - 确认学生身份
   - 记录离校时间

   - 出校时间分布
   - 异常情况提醒

2. **月度报告**
   - 月度出校趋势

   - 频繁出校学生名单

## 总结

本系统提供了完整的校园安全出入管理解决方案，涵盖了所有主要的访问场景，并通过多重验证和审批流程确保校园安全。系统具有良好的扩展性和可维护性，能够适应不同规模学校的需求。
