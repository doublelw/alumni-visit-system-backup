## 🔧 创建测试账户脚本

### 1. 创建预设管理员账户
```python
# 运行: python create_admin_user.py
from backend.app import create_app, db
from backend.app.models import User

app = create_app()
with app.app_context():
    # 检查管理员账户是否存在
    admin = User.query.filter_by(username='admin').first()
    if not admin:
        admin = User(
            username='admin',
            real_name='系统管理员',
            email='admin@school.edu.cn',
            phone='13800000000',
            user_type='admin',
            status='active'
        )
        admin.set_password('admin123')
        db.session.add(admin)
        db.session.commit()
        print("管理员账户创建成功")
    else:
        print("管理员账户已存在")
```

### 完整业务流程测试

1. **校友注册流程**

## 🔍 故障排除

### 常见登录问题

1. **账户被禁用**
   ```python
   # 激活账户脚本
   from backend.app import create_app, db
   from backend.app.models import User

   app = create_app()
   with app.app_context():
       users = User.query.filter_by(status='inactive').all()
       for user in users:
           user.status = 'active'
       db.session.commit()
       print(f"激活了 {len(users)} 个账户")
   ```

2. **密码错误**
   - 确认使用正确的密码格式
   - 使用账户重置密码功能

3. **权限不足**
   - 确认用户类型正确
   - 检查用户状态是否为激活

### 数据库问题

1. **数据重置**
   ```bash
   # 删除测试数据
   python delete_test_data.py
   ```

2. **数据库重建**
   ```bash
   # 重新初始化数据库
   python start.py init
   ```
